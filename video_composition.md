## シーン構成（全10シーン / 計300秒）

### Scene 1: オープニング（0:00〜0:15）15秒

|---|---|

| アイコン | 🎙️ マイクアイコンをタイトル横に装飾的に配置 |

### Scene 3: 前提説明（0:40〜1:05）25秒

**台本**: 「この講座の前提は「ライバー＝売上を作るMC」。出演者ではなく、CV導線を設計して実行する責任者です。」

| 要素 | 内容 |
|---|---|

| フェーズ② (0:52〜1:05) | フローの上に「この導線を"設計して実行"するのがライバー」とオーバーレイテロップ |

| デザイン | フローチャートはカード＋矢印のシンプルなデザイン |

**フローチャート**:
```
[😟 悩み] ──→ [📺 参加] ──→ [💡 提案] ──→ [🎯 CV]
```

### Scene 4: KPI指標（1:05〜1:35）30秒

**台本**: 「この回で意識するKPIの目安は 初級：滞在1分+／コメント率5%+／CVR1%+。数字は"結果"ではなく"改善の手がかり"として使います。」

| 要素 | 内容 |
|---|---|
| ヘッダー | 「📊 初級KPIの目安」タイトル表示 |
| フェーズ① (1:05〜1:15) | 1枚目のKPIカード「⏱️ 滞在時間 1分+」がフリップインで登場 + 意味の補足テロップ「視聴者が離脱せず留まる時間」 |
| フェーズ② (1:15〜1:22) | 2枚目のKPIカード「💬 コメント率 5%+」がフリップイン + 補足「視聴者の参加度を示す指標」 |
| フェーズ③ (1:22〜1:29) | 3枚目のKPIカード「🛒 CVR 1%+」がフリップイン + 補足「購入・成約への転換率」 |
| フェーズ④ (1:29〜1:35) | 3カード並んだ状態で下部に「💡 数字は "結果" ではなく "改善の手がかり"」が強調表示 |
| アニメーション | 各カードは stagger 付きフリップイン、最後のメッセージは spring でポップイン |

**KPIカードデザイン**:
```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│  ⏱️ 滞在時間  │  │  💬 コメント率 │  │  🛒 CVR      │
│    1分+      │  │    5%+       │  │    1%+       │
│ ─────────── │  │ ─────────── │  │ ─────────── │
│ 離脱せず留まる │  │  参加度の指標  │  │ 購入への転換率 │
└──────────────┘  └──────────────┘  └──────────────┘
```

### Scene 5: 3つのポイント概要（1:35〜2:05）30秒

### Scene 7: NG例（2:45〜3:25）40秒

**台本**: 「NG例：『今日はこの商品を紹介します。スペックは…』で開始し、初見が置き去りになる。NG例は"やりがち"ですが、初見が自分ごと化できず、コメントも伸びにくいので注意です。」

| 背景色 | 赤系のソフトグロー（警告的な印象） |
| フェーズ① (2:45〜2:55) | 同様の配信画面風モックアップ → ライバーアイコン |

| フェーズ③ (3:05〜3:15) | バツ印スタンプが吹き出しに押される → 「⚠️ やりがち！」ラベルが点滅 |
| フェーズ④ (3:15〜3:25) | 問題点を箇条書きで表示：「初見が自分ごと化できない」「コメントが伸びにくい」がフェードイン |

## デザインシステム

| 用途 | カラー | 説明 |

| 背景メイン | `#0f0f1a` | ダークネイビー |

| GOOD系 | `#00d2d3` | シアングリーン |

| テキスト白 | `#f0f0f0` | メインテキスト |
| テキスト灰 | `#a0a0b0` | サブテキスト |

### フォント
| 用途 | フォント |
|---|---|
| タイトル・見出し | Noto Sans JP Bold |
| 本文テロップ | Noto Sans JP Medium |
| 数値・KPI | Inter Bold（欧文） |

### アニメーションパターン
| パターン名 | 用途 | 仕様 |
|---|---|---|
| FadeSlideIn | テロップ登場 | opacity 0→1 + translateY 30→0, 0.5秒 |
| PopIn | バッジ・アイコン | scale 0→1.1→1, spring |
| TypeWriter | セリフ表示 | 1文字ずつ表示, 40ms間隔 |
| StampIn | バツ印・チェック | scale 2→1 + rotate, 0.3秒 |
| CardFlip | KPIカード | rotateY 90→0, stagger 0.2秒 |
| PulseGlow | 強調テロップ | opacity 0.7→1→0.7 を繰り返し |

### トランジション
| 遷移 | 方式 |
|---|---|
| Scene間の基本遷移 | クロスフェード（0.5秒） |
| GOOD→NG切替 (S6→S7) | カラーワイプ（緑→赤） |
| まとめ→エンディング | ズームアウト + フェード |

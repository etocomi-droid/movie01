## シーン構成（全10シーン / 計300秒）

### Scene 1: オープニング（0:00〜0:15）15秒

**台本**: 「皆さん、こんにちは。本日の講義、テーマは「ライバーの役割と全体像」についてお話ししていきます。ただカメラの前で配信をするだけでなく、ビジネスとして成果を出すために必要な基本的な考え方を、これからの5分間でギュッと凝縮して分かりやすくお伝えします。それでは、最後までお見逃しなく、一緒に学んでいきましょう！」

| 要素 | 内容 |
|---|---|
| テロップ | タイトル「ライバーの役割と全体像」を大きく中央表示 |
| サブタイトル | 「〜 "売上を作るMC" になるための第一歩 〜」をフェードイン |
| アニメーション | パーティクル背景 → タイトルがスケールアップで登場 → サブタイトルが遅れてフェードイン |
| 背景 | ダークグラデーション + ゆっくり動くパーティクルエフェクト |
| アイコン | 🎙️ マイクアイコンをタイトル横に装飾的に配置 |
| SE | 「ポン」と軽い効果音でタイトル登場を強調 |

### Scene 2: 結論ファースト（0:15〜0:40）25秒

**台本**: 「それでは本題に入ります。まず最初に、最も重要な結論からお話ししておきましょう。ライバーの本質的な役割というのは、単に画面に映る「出演者」ではありません。正しくは、自ら仕掛けを作り、視聴者の具体的な行動を促していく「売上を作るMC」なんです。自分が主役として輝くのではなく、商品やサービスを主役にして場を回していく、プロデューサーのような視点が極めて重要になります。」

| 要素 | 内容 |
|---|---|
| フェーズ① (0:15〜0:25) | 「📌 結論から」ラベル表示 → 「ライバー ＝ 出演者？」のテキストが中央に登場 |
| フェーズ② (0:25〜0:32) | 「出演者」に赤いバツ印がスタンプされる → 打ち消し線アニメーション |
| フェーズ③ (0:32〜0:40) | 画面が分割 → 右側に「売上を作るMC ◎」が緑アクセント付きでスライドイン |
| 背景演出 | 左半面がグレーアウト（NG感）、右半面が明るいアクセントグロー（OK感） |
| 補足テロップ | 画面下部に「CV導線を設計して実行する"責任者"」をフェードイン |

### Scene 3: 前提説明（0:40〜1:05）25秒

**台本**: 「この講座全体を通しての共通認識、すなわち前提は「ライバーイコール、売上を作るMC」であるということです。繰り返しますが、単なる出演者ではありません。ターゲットとなる視聴者の悩みを受け止め、配信に参加してもらい、適切なタイミングで解決策を提案して、最終的な成約に結びつける。この一連の「CV導線」を自ら設計し、現場で実行する責任者だという自覚を強く持ちましょう。」

| 要素 | 内容 |
|---|---|
| フェーズ① (0:40〜0:52) | キーワード図解：「悩みを持つ視聴者」→「配信に参加」→「商品を提案」→「CV（成約）」の流れをフローチャートでアニメーション表示 |
| フェーズ② (0:52〜1:05) | フローの上に「この導線を"設計して実行"するのがライバー」とオーバーレイテロップ |
| アニメーション | フローチャートの各ノードが左から順にポップイン → 矢印が伸びる → 全体をハイライト |
| デザイン | フローチャートはカード＋矢印のシンプルなデザイン |

**フローチャート**:
```
[😟 悩み] ──→ [📺 参加] ──→ [💡 提案] ──→ [🎯 CV]
```

### Scene 4: KPI指標（1:05〜1:35）30秒

**台本**: 「次に、意識すべきKPIの目安についてです。初級レベルでは、まず平均滞在時間1分以上、コメント率5パーセント以上、そして成約率1パーセント以上を目指しましょう。これらの数字は単なる「結果」ではありません。どこを直せばもっと良くなるかを見極めるための、大切な「改善の手がかり」なんです。」

| 要素 | 内容 |
|---|---|
| ヘッダー | 「📊 初級KPIの目安」タイトル表示 |
| フェーズ① (1:05〜1:15) | 1枚目のKPIカード「⏱️ 滞在時間 1分+」がフリップインで登場 + 意味の補足テロップ「視聴者が離脱せず留まる時間」 |
| フェーズ② (1:15〜1:22) | 2枚目のKPIカード「💬 コメント率 5%+」がフリップイン + 補足「視聴者の参加度を示す指標」 |
| フェーズ③ (1:22〜1:29) | 3枚目のKPIカード「🛒 CVR 1%+」がフリップイン + 補足「購入・成約への転換率」 |
| フェーズ④ (1:29〜1:35) | 3カード並んだ状態で下部に「💡 数字は "結果" ではなく "改善の手がかり"」が強調表示 |

### Scene 5: 3つのポイント概要（1:35〜2:05）30秒

**台本**: 「ここからは、ライバーとして成功するためのポイントを3つの柱に分けて解説していきます。一つ目は、主役としての振る舞いを変えること。二つ目は、事前の導線作り。そして三つ目は、数値に基づいた運営責任です。この3つを意識するだけで、あなたの配信の質は劇的に変わります。」

| 要素 | 内容 |
|---|---|
| ヘッダー | 「🔑 3つのポイント」タイトル |
| フェーズ① (1:35〜1:45) | ポイント①「🎤 ライバーは出演者ではなく "売上を作るMC" である」がスライドダウンで登場 → 登場後にアイコンが軽くバウンス |
| フェーズ② (1:45〜1:55) | ポイント②「🗺️ 導線設計（悩み→参加→提案）を先に作る」が同様に登場 → フロー矢印のミニアニメがカード内で再生 |
| フェーズ③ (1:55〜2:05) | ポイント③「📊 場の運営（コメント・温度）と数値改善の責任を持つ」が登場 → グラフアイコンが軽く上昇アニメ |
| 背景 | 3ポイントが揃った後、全体にうっすらハイライトのパルスアニメーション |

**レイアウト**:
```
🔑 3つのポイント

  ① 🎤  ライバーは出演者ではなく"売上を作るMC"である

  ② 🗺️  導線設計（悩み→参加→提案）を先に作る

  ③ 📊  場の運営と数値改善の責任を持つ
```

### Scene 7: NG例（2:45〜3:25）40秒

**台本**: 「一方で、初心者の方が特によくやってしまいがちなNG例がこちらです。『さあ、今日はこの新しい商品を紹介していきます。まずスペックから見ていきましょう。重さは…』と、いきなり一方的な説明を始めてしまうパターンです。これでは初見の視聴者が置き去りになり、内容を自分ごととして捉えることができません。結果として、コメントも盛り上がらず、成約にも繋がらないため、注意が必要です。」

| 要素 | 内容 |
|---|---|
| トランジション | Scene 6 からカラーワイプ（緑→赤）で切り替え |
| バッジ | 画面左上に「❌ NG」バッジ（赤系）がポップイン |
| 背景色 | 赤系のソフトグロー（警告的な印象） |
| フェーズ① (2:45〜2:55) | 同様の配信画面風モックアップ → ライバーアイコン |
| フェーズ② (2:55〜3:05) | 吹き出しにセリフ表示：「今日はこの商品を紹介します。スペックは…」→ 視聴者アイコンが「😴💤」に変化 |
| フェーズ③ (3:05〜3:15) | バツ印スタンプが吹き出しに押される → 「⚠️ やりがち！」ラベルが点滅 |
| フェーズ④ (3:15〜3:25) | 問題点を箇条書きで表示：「初見が自分ごと化できない」「コメントが伸びにくい」がフェードイン |

**GOOD / NG の対比まとめ**:
```
Scene 6 (GOOD)              Scene 7 (NG)
┌────────────────┐         ┌────────────────┐
│ ✅ GOOD         │    →    │ ❌ NG           │
│ 緑アクセント     │         │ 赤アクセント     │
│ ターゲット明確   │         │ 一方的な紹介     │
│ 視聴者巻き込み   │         │ 初見置き去り     │
└────────────────┘         └────────────────┘
```

### Scene 8: リスク注意（3:25〜3:55）30秒

**台本**: 「また、健全な運営のためにリスク管理も欠かせません。規約違反となる外部サイトへの不適切な誘導や、視聴者・他者との喧嘩、そして薬機法などに触れる過剰な断言は絶対にNGです。これらは一度でも起こすと取り返しのつかない炎上や法的リスクに繋がります。「知らなかった」では済まされないので、最初から強く意識しておきましょう。」

| 要素 | 内容 |
|---|---|
| ヘッダー | 「⚠️ リスク注意」が警告色で登場（画面上部にバナー風） |
| フェーズ① (3:25〜3:35) | 1つ目「🚫 外部誘導NG」がスライドインで登場 + 補足「他サイトへの不正な誘導は規約違反・炎上リスク」 |
| フェーズ② (3:35〜3:43) | 2つ目「🚫 喧嘩NG」がスライドイン + 補足「視聴者・他配信者との口論は即炎上の原因に」 |
| フェーズ③ (3:43〜3:50) | 3つ目「🚫 断言しすぎNG」がスライドイン + 補足「効果の過剰な断言は法的リスクにもなる」 |
| フェーズ④ (3:50〜3:55) | 3項目が並んだ状態で全体を赤枠で囲む → 「最初から意識しておく！」を強調テロップ |
| 背景 | ダークベース + オレンジ/赤のウォーニンググロー |

**レイアウト**:
```
⚠️ リスク注意 ─── 最初から意識！
┌──────────────────────────────────┐
│ 🚫 外部誘導NG    → 規約違反・炎上   │
│ 🚫 喧嘩NG       → 即炎上の原因     │
```

### Scene 9: ミニドリル（3:55〜4:35）40秒

| フェーズ④ (4:28〜4:35) | 「📱 録画 or 📝 メモに残して → 相互チェックに使おう」が下部にフェードイン |

### Scene 10: まとめ＋エンディング（4:35〜5:00）25秒

**台本**: 「最後に今日の内容をまとめましょう。ライバーの真の役割は、単に目立ったり人気者になったりすることではなく、一人の「売上を作るMC」として戦略的に場を動かすことです。この基本的な考え方をしっかりと自分の「型」として落とし込み、何度でも再現できるようになれば、あなたの配信結果は確実に安定し、効果的な改善も進めやすくなります。自信を持って、次のステップへと進んでいきましょう。お疲れ様でした！」

| 要素 | 内容 |
|---|---|
| フェーズ① (4:35〜4:42) | 「📌 まとめ」ヘッダー表示 |
| フェーズ② (4:42〜4:50) | 結論の再掲を大きく中央表示：「ライバー ＝ 売上を作るMC」（Scene 2 と同じデザインで統一感） |
| フェーズ③ (4:50〜4:55) | 補足テロップ：「この型で "再現" できれば → 配信は安定 → 改善もしやすい」が段階的にフェードイン |
| フェーズ④ (4:55〜5:00) | 「Next → 次へ進みましょう」テロップ → 全体がズームアウト+フェードアウト |
| 背景 | オープニングと同じダークグラデーション＋パーティクルに戻る（ループ感） |
